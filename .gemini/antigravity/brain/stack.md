# Tech Stack
- Frontend: Vanilla HTML/JS with Tailwind CSS tokens
- API: FastAPI (Python)
- Voice Engine: LiveKit Agents Framework
- DB: PostgreSQL via `psycopg2`
- Models: gpt-4.1-mini, Sarvam STT/TTS
- Justification: LiveKit provides the lowest latency WebRTC transport natively compatible with Python async execution.
